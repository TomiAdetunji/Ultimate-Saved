# Ultimate-Saved
Welcome to the Ultimate Saved, where the goal here is to keep you on top of all the saved tutorials or inspiration you come across on Instagram, TikTok, Youtube, Pinterest, and more by notifying you of a different video you can follow every day. Not only will this be a big saved folder for all your social media sites but also a way to log and detail your experience with each tutorial or project. Allowing you to reflect on the project and what you learned and wthere you'd like to do it again and more. 

This project was made mostly in TYpescript with the help of NextJS framework to build the website. On this day, 3/16/2026, so far I have the user submitting links working, Google Gemini creating data about the links/content to display on the link cards (social media site, categories they could fall into, A likely title and a thumbnail image), these links are saved in local storage which needs to be moved to Supabase, and just got a lightbox working when the cards are clicked. The lightbox currently just enlarges the thumbnail image and display the title. Hopefully I can replace the thumbnail with the actual video playing and a more in depth description of the video.
